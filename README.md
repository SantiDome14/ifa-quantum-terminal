**IFA QUANTUM** es una terminal profesional de gestión de activos y optimización de portafolios basada en la Teoría Moderna de Portafolios (Markowitz). Diseñada para inversores institucionales y analistas cuantitativos, esta herramienta permite realizar simulaciones de Monte Carlo, backtesting histórico y análisis profundo de riesgos (VaR, CVaR)

---

## Funcionalidades Principales

* **Motor de Optimización Markowitz:** Cálculo de la Frontera Eficiente para maximizar el Ratio de Sharpe
* **Análisis de Riesgo Avanzado:** Cálculo automático de métricas institucionales como VaR (95%), CVaR (Expected Shortfall) y Ratio Sortino
* **Simulaciones de Monte Carlo:** Ejecución de miles de escenarios probabilísticos para validar la robustez del portafolio
* **Datos de Mercado en Vivo:** Conexión directa a Yahoo Finance para precios ajustados y divisas en tiempo real
* **Reportes Exportables:** Descarga de estructuras de portafolio y series de precios en formato Excel (.xlsx)
